# Airline Performance Dashboard

## Project Overview

This project analyzes airline operational performance using Microsoft Excel.

The dashboard tracks:

- Revenue
- Passenger Trends
- Route Profitability
- Load Factor
- Aircraft Utilization

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## KPIs

- Total Revenue
- Total Passengers
- Average Load Factor
- Top Performing Route
- Least Profitable Route



## Key Insights

- Route A generated the highest revenue.
- Passenger demand increased in Q3.
- Load Factor remained above 82%.
- Domestic routes contributed most revenue.
